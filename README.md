# inventoryjs

## How to run

- Kubernetes

  - Build.sh

    - Arguments:
      - `-b true` to build
      - `-p true` to push
      - ex: `./build.sh -b=true -p=true`

  - Deploy.sh

    - Arguments:
      - `-u username`
      - `-p password`
      - `-h host`
      - `-d database`
      - `-p port`

- Docker compose
